# Sistema de Mensagens com Flask e Streamlit

Este é um sistema simples de mensagens que utiliza Flask como servidor API e Streamlit para a interface do usuário.

## Pré-requisitos

- Python 3.x instalado
- Instalação do RabbitMQ (ou outro Message Oriented Middleware de sua escolha)
